 GlowMatch

Web app recommending foundation & lipstick shades for South Asian skin tones. Uses HSV color space, K-Means clustering & CIELAB Delta-E algorithms. 700+ product database. Built with HTML/CSS/JS. Final year project at Women University Swabi.

 How It Works
1. Upload facial photo
2. Skin tone detection via HSV color space
3. K-Means clustering finds dominant tone
4. CIELAB Delta-E matches against 700+ products
5. Displays top 5 foundation & lipstick matches

 Files
- `index.html` — Main app
- `app.js` — Application logic
- `colorscience.js` — Color algorithms
- `productdatabase.js` — 700+ products
- `style.css` — Styling
- `login.html`, `signup.html`, `welcome.html` — Auth pages
